Descrição do projeto:

O Projeto é uma web page de slider que apresenta produtos. A página segue uma sua 
alteração sendo feita pelo uso dos botões laterais, com determinada ação de click
o próximo produto é apresentando.
----------------------------------------------------------------
Instruções de como executar a aplicação:

-clonar o repositório
-Compilação do TypeScript
